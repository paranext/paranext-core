    Interface NetworkEventHandlers

    Event data types for each network event available on the papi. Each extension can extend this interface to add events it emits with papi.network.createNetworkEventEmitter.

    Note: Event names must consist of two strings separated by at least one period. We recommend one period and lower camel case in case we expand the api in the future to allow dot notation.

    An extension can extend this interface to add types for the network events it emits by adding the following to its .d.ts file:

    Example

    declare module 'papi-shared-types' {
      export interface NetworkEventHandlers {
        'myExtension.onSomethingChanged': { someData: string };
      }
    }
