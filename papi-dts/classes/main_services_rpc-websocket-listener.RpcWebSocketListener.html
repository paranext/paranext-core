    Class RpcWebSocketListener

    Owns the WebSocketServer that listens for clients to connect to the web socket. When a client connects, an RpcServer is created in this same process to service that connection.

    Also owns a map of all registered methods tied to the IRpcHandler that knows how to respond to those methods. Methods registered within "main" will be tied to this class. Methods registered by other processes will be tied to the RPCServer that services the connection for that other process.

    Created by the main process on start up when the network service initializes
