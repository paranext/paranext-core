# Theming

A guide to theming components using the shadcn design system in this library: token usage, theme
switching in Storybook, CSS variables, and how they relate to the Paranext application. By default we
stick to the existing shadcn-style tokens defined in `index.css` and consumed through Tailwind.

## Theme matrix

Compares Platform and Paratext palettes in one view using scoped wrappers. Useful for visual review
of theme tokens. For the live token table for the toolbar-selected theme, see [Guides / Theme
Colors](https://paranext.github.io/paranext-core/platform-bible-react-storybook/?path=/docs/guides-theme-colors--docs).
