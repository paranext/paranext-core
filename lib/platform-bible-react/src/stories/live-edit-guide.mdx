# Live code editing guide

This Storybook includes live code editing capabilities powered by `storybook-addon-code-editor`. You can edit React TypeScript code in real-time using the Live Code Editor tab in the Storybook Panel and see changes immediately on the canvas.

## How to Use

1. **Navigate to any Live Edit enabled story** in this section
2. **Look for the live code editor** - It will appear as a a tab in the bottom section.
3. **Edit the code** directly in the editor
4. **See changes instantly** on the canvas above

## Editing code
Changes are temporary and won't affect the actual codebase.
* Use VS Code shortcuts
* **Ctrl/Cmd+Z** to undo changes
* **Ctrl/Cmd+S** to format code
