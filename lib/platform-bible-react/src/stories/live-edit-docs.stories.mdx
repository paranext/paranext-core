# Live Code Editing

This Storybook includes live code editing capabilities powered by `storybook-addon-code-editor`. You can edit React/TypeScript code in real-time and see changes immediately in the preview.

## How to Use

1. **Navigate to any Live Edit story** in this section
2. **Look for the code editor** - It will appear as an editor panel or section
3. **Edit the code** directly in the Monaco editor (VS Code-like experience)
4. **See changes instantly** reflected in the component preview

## Features

- ‚úÖ **Monaco Editor** - VS Code's editor with familiar shortcuts
- ‚úÖ **TypeScript Support** - Full syntax highlighting and IntelliSense
- ‚úÖ **Real-time Updates** - Changes appear immediately
- ‚úÖ **React Hooks** - useState, useEffect, and other hooks work out of the box
- ‚úÖ **Component Library** - Your Button, Input, and other components are available

## Quick Tips

### Editing Code

- Use **Ctrl/Cmd+Z** to undo changes
- **Ctrl/Cmd+S** to format code
- All standard VS Code shortcuts work

### Common Patterns

```tsx
// Add state
const [value, setValue] = useState('initial');

// Add event handlers
onClick={() => console.log('clicked')}

// Use components
<Button variant="destructive">Click me</Button>
<Input placeholder="Type here..." />
```

### What's Available

- **React** - Automatically imported (useState, useEffect, etc.)
- **Button** - All variants: default, destructive, outline, secondary, ghost, link
- **Input** - Text inputs with full event handling
- **Standard HTML** - All HTML elements and inline styles

## Best Practices

‚úÖ **Start simple** - Begin with BasicExample to get familiar
‚úÖ **Experiment freely** - Changes are temporary and won't affect the actual codebase
‚úÖ **Copy useful code** - When you create something useful, copy it for your projects
‚úÖ **Try different patterns** - Test controlled vs uncontrolled components, different event handlers, etc.
