# PR Screenshots for UI Changes

A PR that changes the UI — React components (`*.component.tsx`), web views
(`*.web-view.tsx`), or styling (`*.scss` / `*.css`) — should include before/after
screenshots in its description, or a short clip for interactions and animations, so a
reviewer can see the visual effect without building the branch.

See [media.md](media.md) for where to host those images (not the code repo).
